# SMARTBOLI

- **Idea**: Smartboli es un proyecto personal que consiste en una un boligrafo inteligente con aplicacion que pueda tomar notas fisicas en papel, poder almacenar esta informacion de manera digital, poder editarla y compartirla mediante enlace.
Dentro del desarrollo del la plataforma o pagina web el usuario podra ver todas sus notas guardadas en el boligrafo, adiciona a esto la pagina web tendra un motor de IA que podra ayuda al cliente ampliar los conocimientos en los temas que se encuentren en las notas, brindando material visual, audios, enlaces, paginas web relacionadas y resumenes, adicional a esto contara con una seccion de cursos relacionados al tema, ejercicios practicos y didacticos brindados por el ajente de los cuales se podra avanzar tema a tema y se llevara un seguimiento del progreso del usuario, de igual manera se brindaran consejos y recomendaciones personalizados para cada usuario.

## Descripción

- Smartboli busca implementar una experiencia de usuario integral, que combine la tradicionalidad del papel con la innovación de la tecnología, permitiendo a los usuarios guardar, editar y compartir sus notas de manera digital y mejorar la experiencia de aprendizaje.
